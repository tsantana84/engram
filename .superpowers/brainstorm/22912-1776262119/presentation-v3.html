03 · How It Works

Automatic. Invisible.
Always on.

01
Engineer works normally in Claude Code
No change to workflow. Engram runs silently in the background, watching every tool use, file edit, and decision.
PostToolUse hook
02
Session is compressed into observations
At the end of each session, an AI agent summarizes what happened — bugs fixed, decisions made, code written — into structured, searchable records.
AI compression · ~71% token savings
03
Observations sync to a shared database
Every observation is pushed to a central Supabase database. All agents on all machines contribute to — and can search — the same knowledge base.
Vercel + Supabase · always available
04
Next session starts with context already loaded
When a new session opens, relevant past work is injected automatically. The AI already knows what was done. No re-explaining required.
SessionStart hook · instant context
