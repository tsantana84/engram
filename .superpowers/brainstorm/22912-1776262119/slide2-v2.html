01 · The Problem

AI agents have partial memory.
The important parts are missing.

Tools exist to give AI some context — but they're manual, static, and shallow. The rich work history that makes an engineer truly effective? That's still lost between sessions.

✓ What exists today
📄
CLAUDE.md / project docs
Manual, static, project-scoped — you write it, you maintain it
💡
Claude's built-in memory
High-level preferences — not work history
🗒️
Project READMEs & wikis
Architecture docs, if they're up to date
✗ What's still missing
🔴
Automatic session history
What was actually done, decided, and discovered last week
🔴
Cross-session search
"How did we solve this before?" — no answer without manual notes
🔴
Shared team knowledge
What your colleague's AI learned yesterday isn't available to yours today
The gap isn't total amnesia — it's that the useful memory, the lived work history, is still lost every session.
