What's the delivery format?

Shapes structure, interactivity, and depth

A

Live talk / demo session

Slides + live terminal demo. 15–30 min. Speaker walks through install, shows sync working in real time.

B

Self-serve HTML presentation

Like the existing leadership/team decks. Developer opens the file, reads through at their own pace. No speaker needed.

C

Written guide / doc page

Prose + code blocks. Deep technical walkthrough. More like a developer onboarding doc than a presentation.
